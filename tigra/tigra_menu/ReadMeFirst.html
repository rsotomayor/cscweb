How to start with Tigra Menu?

The easiest way to configure the menu is using your local copy of the Tigra Menu Builder. The builder is the web based cross-platform application that helps you to visually create and customize the basic menu without learning about the menu configuration structures. CLICK HERE TO BEGIN >>

For the most advanced setup you may need to manually adjust the configuration files. This can be done by following these steps:

  • Review product documentation to learn about the configuration structures of Tigra Menu
  • Review the sample configurations provided in this package and select one which is the closest to your design idea. Alternatively you can use the configuration created with the menu builder as the starting point
  • Copy files to separate directory and modify the configuration structures according to your needs
  • If you have any problems or questions, check out the support forum
  • As you have your menu set up, replace menu.js with its size optimized version (10KB>>3.5KB) to reduce download time and save network traffic. The file can be found in the directory "compressed"

Here are some suggestions:

  • Keep original files from the distribution package unchanged. Extract any files from zip file to separate directory before editing them
  • Modify script configuration step-by-step checking if the menu still works in the browser after each modification. This will help you to rollback the last change if it was erroneous
  • Check out product page periodically. We keep working to bring new versions of the product and more support materials for your convenience
  • Subscribe to our RSS feed for latest news, product updates and special offers

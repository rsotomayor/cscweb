Welcome to Tigra Menu Builder
Description Tigra Menu Builder is free DHTML application that helps create professional level javascript navigation systems for the web sites. The resulting solution relies on one of the Tigra Menu components.
System Requirements This builder is platform independent DHTML application. All it needs to run is the compatible browser. It will work in most of the modern browsers but yours doesn't seem to be one of them. We suggest upgrading it to some recent version. Please note that the browser compatibility of the generated menu system only depends on the menu product that you choose. You can find the browser compatibility information at the product description pages in "mode" section of this application.
Skills Requirements All the cross-browser JavaScript programming is hidden under the hood of the Tigra Menu components so no programming skills are needed. The menus are configured via text based configuration files and this builder helps to create them without understanding and following their syntax. The whole process is intuitive and well commented. At the end you'll receive ready to copy-n-paste fragments of code together with clear instructions on how and where to place them in the HTML code of your page(s).

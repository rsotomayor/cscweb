Level Style

Style select box specifies which of the pre-designed style templates should be applied to the menu items in currently selected level. Click "styles" navigation tab in top pane to manage and edit the style templates.

Note that style setting is shared among all menu items on the same level/depth. It is possible to apply different settings to each item individually using the HTML formatting in the caption of the menu item.

By default the style is inherited from the parent level

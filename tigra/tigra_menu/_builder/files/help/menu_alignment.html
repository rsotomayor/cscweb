Menu alignment

The feature that allows to keep menu aligned horizontally (left, right, center) and/or vertically (top, bottom, center) on the page regardless of the browser's window size. Offsets can be defined to adjust menu position (i.e. 40px from the right edge of the document or 100px shift to the left from the center). Without this feature menu position is always specified absolutely from the top left edge of the document.

Level Layout

Layout select box specifies the way the items are aligned withing the menu block. Most common options are vertical (top to bottom) or horizontal (left to right) layout of the root level and vertical (top to bottom) layout of all sub-levels. Other options are also supported.

In Tigra Menu and Tigra Menu PRO the layout is defined by Horizontal Offset and Vertical Offset parameters of the menu level, so changing the layout selection will update these fields and wise versa.

Note that layout setting is shared among all menu blocks on the same level/depth. In Tigra Menu PRO it is possible to redefine the layout for each menu block using Horizontal Offset and Vertical Offset parameters of the items.

By default the layout is inherited from the parent level

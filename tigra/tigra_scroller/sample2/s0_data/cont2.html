Features:
  • Unlimited number of scroller instances on the page, each independently configurable
  • Scroller look is configurable via CSS
  • Any HTML code can be scrolled including tables, images etc
  • External HTML files can be used to get scrolling content
  • Multiple scroll items
  • Adjustable scrolling speed
  • Either vertical or horizontal scroll direction
  • Customizable pause time before and after an item
  • Scroll modes: auto and on-demand (when control elements are mouseovered)
  • Pause on mouseover in auto mode
  • Works in both frames and iframes
  • Configuration is optimized for dynamic generation by server side script
  • All popular browsers supported
  • Future browser's versions ready
  • Easy setup - no programming skills required. Online documentation is available.
  • The script is free for any applications

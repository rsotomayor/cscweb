Tigra Hints v2.0 - Welcome Page

Thank you for downloading Tigra Hints - free JavaScript component by Softcomplex designed to display pop-up messages and forms in HTML documents.

Contents of this package

The zip package you downloaded includes the series of the working samples demonstrating different ways in which the script can be used. You can find these samples in the directories beginning with the word "sample".

The script's code resides in the file called tigra_hints.js. No changes are normally required in that file. All the configuration is done via the configuration variables. If you decide to modify the script's file please make sure to leave the product information at the top of the file unchanged, you can add your comments below it.

The provided samples use well formatted and commented version of the software. If you don't plan to make any changes to the core script we recommend using the compressed version of it. This will speed up the page loading and decrease the bandwidth usage. You can find the compressed version in the directory called "compressed".

Technical Support

If you have any questions about the script please make sure to first review the product documentation. It has detailed information about all aspects of the script configuration as well as useful tips, trick and troubleshooting recommendations.

If you still need help feel free to ask for it in the public forum. When posting please make sure to comply with the forum rules and consider privacy issues associated with public postings.

If you believe your issue is very specific to your particular situation or if you need to provide privacy sensitive information then you can use one of the direct contact options listed in the support section of our website.

Feedback and Contributions

Tigra Hints is full featured free product. As the experiment, we will not spin off the more advanced paid version of this software in order to support its development. Instead we'll be implementing all new requested features in this free script while funding the effort solely from your voluntary donations. So if you find Tigra Hints useful we kindly ask you to make a modest ([$5] [$10]) contribution which should be sufficient to keep this project alive.
